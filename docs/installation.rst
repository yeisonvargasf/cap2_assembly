============
Installation
============


Stable release
--------------

To install CAP2 Assembly, run this command in your terminal:

.. code-block:: console

    $ pip install cap2_assembly

This is the preferred method to install CAP2 Assembly, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for CAP2 Assembly can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/lauren-mak/cap2_assembly

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/lauren-mak/cap2_assembly/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/lauren-mak/cap2_assembly
.. _tarball: https://github.com/lauren-mak/cap2_assembly/tarball/master
